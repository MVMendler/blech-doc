Embedded real-time programming

{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Blech is a programming language for embedded, reactive, real-time critical software. The German word _Blech_ roughly translates to _bare metal_. Blech programs compile to C89 and can run directly on _the Blech_ on pretty much any embedded device. {{% /blocks/lead %}} {{< blocks/section color="white" >}}
