About us

This website makes the Blech language documentation, the Blech compiler, and the Blech tools accessible.

  • The website is a GitHub project page boschresearch.github.io/blech-doc .
  • The GitHub organization is boschresearch
  • The GitHub project for this website is blech-doc .
  • The website is hosted on GitHub’s github.io domain.

The website is accessible via

  1. the subdomain http://www.blech-lang.org,
  2. the apex domain http://blech-lang.org, and
  3. the GitHub pages site https://boschresearch.github.io/blech-doc.

The apex domain and the GitHub pages site redirect to the subdomain.

The website uses

  • GitHub : for hosting and direct access to documentation and the Blech project repositories.
  • Google Analytics : on the basis of your opt-in consent, for improving your experience on our website.
  • Google Custom Search : for in-site search queries.
